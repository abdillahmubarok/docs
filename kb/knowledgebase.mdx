## 🚀 Memulai Perjalanan Anda

### 📝 Cara Mendaftar Akun Baru

#### Langkah 1: Mengisi Form Pendaftaran

1. **Buka Website Resmi**
   - Kunjungi https://mubarokah.id
   - Klik tombol **"Daftar"** di pojok kanan atas
2. **Isi Informasi Dasar**

   ```
   ✅ Nama Lengkap: Masukkan nama sesuai KTP
   ✅ Email: Gunakan email aktif yang mudah diakses
   ✅ Nomor HP/WhatsApp: Pilih kode negara (+62) lalu nomor
   ✅ Password: Minimal 8 karakter dengan kombinasi huruf, angka, simbol
   ✅ Konfirmasi Password: Ulangi password yang sama
   ```
3. **Kekuatan Password - Panduan Visual**
   - 🔴 **Lemah**: hanya huruf kecil (contoh: mubarokah)
   - 🟡 **Sedang**: huruf \+ angka (contoh: mubarokah123)
   - 🟢 **Kuat**: huruf \+ angka \+ simbol (contoh: Mubarokah123\!)
4. **Klik "Daftar"**
   - Tunggu proses validasi
   - Jika ada error, perbaiki sesuai petunjuk

#### Langkah 2: Memilih Metode Verifikasi

Setelah berhasil mendaftar, Anda akan diarahkan ke halaman pemilihan verifikasi:

**🔄 Pilihan A: Verifikasi Email**

- ✅ Lebih aman dan permanen
- ✅ Dapat diakses dari mana saja
- ✅ Tidak tergantung koneksi seluler
- 📧 Link verifikasi dikirim ke email

**🔄 Pilihan B: Verifikasi WhatsApp**

- ✅ Lebih cepat (instant)
- ✅ Familiar untuk pengguna Indonesia
- ✅ Langsung dapat kode 6 digit
- 📱 Kode dikirim via WhatsApp

> 💡 **Rekomendasi**: Pilih verifikasi email untuk keamanan jangka panjang

#### Langkah 3: Proses Verifikasi

**Jika Memilih Email:**

1. Buka email Anda (cek inbox dan spam)
2. Cari email dari "Mubarokah ID Security Team"
3. Klik link "Verify Email Address"
4. Otomatis diarahkan ke halaman sukses

**Jika Memilih WhatsApp:**

1. Tunggu pesan WhatsApp (max 2 menit)
2. Salin kode 6 digit
3. Masukkan kode di halaman verifikasi
4. Klik "Verifikasi"

#### Langkah 4: Login Pertama

1. Kembali ke halaman login
2. Masukkan email dan password
3. Klik "Masuk"
4. Selamat\! Akun sudah aktif

## 🔐 Mengamankan Akun dengan 2FA

**Two-Factor Authentication (2FA)** adalah lapisan keamanan tambahan yang **SANGAT DIREKOMENDASIKAN**. Tanpa 2FA, akun Anda 10x lebih rentan diretas\!

### 🛡️ Mengapa 2FA Penting?

- 🚫 **Melindungi dari pencurian password**
- 🚫 **Mencegah akses tidak sah meski password bocor**
- 📊 **Mengurangi risiko peretasan hingga 99.9%**
- ✅ **Memberikan notifikasi login yang mencurigakan**

### 📱 Metode 1: Setup 2FA dengan Aplikasi Authenticator

#### Persiapan

1. **Download Aplikasi Authenticator** (pilih salah satu):
   - 🥇 **Google Authenticator** (paling populer)
   - 🥈 **Authy** (backup cloud)
   - 🥉 **Microsoft Authenticator** (integrasi Microsoft)

#### Langkah Setup

1. **Masuk ke Pengaturan Keamanan**

   ```
   Dashboard → Profil → Keamanan → Setup Two-Factor Authentication
   ```
2. **Pilih "Setup Authenticator"**
   - Klik opsi Authenticator App
   - Muncul halaman dengan QR Code
3. **Scan QR Code**
   - Buka aplikasi authenticator
   - Pilih "Add Account" atau ikon "\+"
   - Scan QR code di layar
   - **Alternatif**: Masukkan kode manual jika tidak bisa scan
4. **Verifikasi Setup**
   - Aplikasi akan menampilkan kode 6 digit
   - Masukkan kode di website
   - Klik "Verifikasi & Aktifkan"
5. **Simpan Recovery Codes** ⚠️ **SANGAT PENTING\!**
   - Download atau print 8 recovery codes
   - Simpan di tempat aman (tidak di HP/komputer)
   - Kode ini untuk akses darurat jika HP hilang

### 📧 Metode 2: Setup 2FA dengan Email

#### Langkah Setup

1. **Pilih "Setup Email"**
   - Di halaman 2FA setup, pilih Email
   - Sistem kirim kode ke email terdaftar
2. **Cek Email**
   - Buka email dari "Mubarokah ID Security Team"
   - Cari kode 6 digit
   - **Jika tidak ada**: Cek folder spam/junk
3. **Masukkan Kode**
   - Input kode 6 digit di website
   - Klik "Verifikasi & Aktifkan"
4. **Simpan Recovery Codes**
   - Download 8 recovery codes
   - Simpan dengan aman

### 🔄 Cara Menggunakan 2FA Saat Login

**Setelah 2FA aktif, setiap login akan memerlukan:**

1. **Login Normal**
   - Masukkan email \+ password
   - Klik "Masuk"
2. **Verifikasi 2FA**
   - **Jika pakai Authenticator**: Buka app, masukkan kode 6 digit
   - **Jika pakai Email**: Cek email, masukkan kode 6 digit
3. **Akses Berhasil**
   - Masuk ke dashboard

### 🆘 Menggunakan Recovery Code

**Jika HP hilang atau tidak bisa akses authenticator:**

1. Di halaman 2FA, klik "Use Recovery Code"
2. Masukkan salah satu dari 8 recovery codes
3. **PENTING**: Setiap code hanya bisa dipakai 1x
4. Setelah login, segera setup 2FA ulang

### ❌ Menonaktifkan 2FA

> ⚠️ **Peringatan**: Menonaktifkan 2FA mengurangi keamanan akun\!

1. **Masuk ke Pengaturan**

   ```
   Profil → Keamanan → Two-Factor Authentication
   ```
2. **Klik "Disable 2FA"**
   - Masukkan password untuk konfirmasi
   - Klik "Disable Two-Factor Authentication"
3. **2FA Berhasil Dinonaktifkan**
   - Akun kembali ke keamanan dasar
   - Recovery codes dihapus

## 👤 Mengelola Profil & Akun

### ✏️ Mengupdate Informasi Profil

#### Informasi Dasar

1. **Akses Edit Profil**

   ```
   Dashboard → Profil → Edit Profile
   ```
2. **Update Data Pribadi**
   - 📝 **Nama Lengkap**: Sesuaikan dengan dokumen resmi
   - 👤 **Username**: Untuk login dan URL profil (contoh: @johndoe)
   - 📧 **Email**: Hati-hati mengubah ini (perlu verifikasi ulang)
   - 📞 **No. HP/WhatsApp**: Untuk notifikasi dan recovery
3. **Informasi Tambahan**
   - 📖 **Bio**: Deskripsi singkat (max 200 karakter)
   - 🏙️ **Tempat Lahir**: Kota kelahiran
   - 📅 **Tanggal Lahir**: Format DD/MM/YYYY
   - ⚧️ **Jenis Kelamin**: Laki-laki/Perempuan
   - 🏠 **Alamat**: Alamat lengkap
4. **Simpan Perubahan**
   - Scroll ke bawah
   - Klik "Update Profile"
   - Tunggu konfirmasi sukses

#### Mengupload Foto Profil

1. **Klik Area Foto**
   - Di halaman edit profil, klik lingkaran foto
   - Atau klik "Change Photo"
2. **Pilih File**
   - Format: JPG, PNG, GIF
   - Ukuran max: 2MB
   - Resolusi ideal: 200x200px
3. **Crop & Adjust**
   - Sesuaikan posisi crop
   - Pastikan wajah terlihat jelas
   - Klik "Save"

### 🔑 Mengelola Password

#### Mengubah Password

1. **Masuk ke Keamanan**

   ```
   Profil → Keamanan → Password Security
   ```
2. **Update Password**
   - **Current Password**: Password lama
   - **New Password**: Password baru (min 8 karakter)
   - **Confirm Password**: Ulangi password baru
3. **Tips Password Kuat**

   ```
   ✅ Minimal 8 karakter
   ✅ Kombinasi huruf besar & kecil
   ✅ Mengandung angka
   ✅ Mengandung simbol (!@#$%^&*)
   ❌ Jangan gunakan info personal
   ❌ Jangan gunakan password yang sama di situs lain
   ```

#### Reset Password (Lupa Password)

1. **Di Halaman Login**
   - Klik "Forgot Password?"
   - Masukkan email akun
   - Klik "Send Password Reset Link"
2. **Cek Email**
   - Buka email dari Mubarokah ID
   - Klik link reset password
   - **Valid selama**: 1 jam
3. **Buat Password Baru**
   - Masukkan password baru 2x
   - Klik "Reset Password"
   - Otomatis masuk ke akun

### ⚠️ Menghapus Akun

> 🚨 **PERHATIAN**: Tindakan ini **PERMANEN** dan **TIDAK BISA DIBATALKAN**

1. **Masuk ke Danger Zone**

   ```
   Profil → Danger Zone → Delete My Account
   ```
2. **Konfirmasi Penghapusan**
   - Baca peringatan dengan teliti
   - Masukkan password
   - Centang "I understand this action cannot be undone"
   - Klik "Delete Account"
3. **Data yang Akan Dihapus**
   - ❌ Semua informasi profil
   - ❌ Session dan login history
   - ❌ Akses ke aplikasi terintegrasi
   - ❌ Recovery codes dan 2FA settings

## 💻 Mengelola Sesi & Keamanan

#### Melihat Sesi Aktif

1. **Akses Session Management**

   ```
   Dashboard → Manage Sessions → Active Devices
   ```
2. **Informasi Setiap Sesi**
   - 🖥️ **Device Type**: Desktop, Mobile, Tablet
   - 🌐 **Browser**: Chrome, Firefox, Safari, dll
   - 🌍 **Location**: Perkiraan lokasi berdasarkan IP
   - 🕐 **Last Activity**: Waktu aktivitas terakhir

   - ⏰ **Login Time**: Kapan pertama kali login

#### Mengenali Sesi Mencurigakan

🚨 **Segera lakukan tindakan jika melihat:**

- 🌍 **Lokasi asing**: Login dari negara/kota yang tidak dikenal
- ⏰ **Waktu aneh**: Aktivitas di jam tidur
- 🖥️ **Device asing**: Perangkat yang bukan milik Anda
- 🌐 **Browser tidak dikenal**: Browser yang tidak pernah Anda pakai

### 🚫 Menutup Sesi

#### Menutup Sesi Tertentu

1. **Pilih Sesi Mencurigakan**
   - Di halaman Active Devices
   - Klik "End Session" pada sesi yang ingin ditutup
2. **Konfirmasi**

   ```
   ⚠️ End Session?
   
   Are you sure you want to end the session on device [Device Name]?
   This device will be logged out from your account.
   
   [ Cancel ]  [ End Session ]
   ```
3. **Hasil**
   - ✅ Sesi segera dihentikan
   - 🚫 Device tersebut harus login ulang
   - 📧 Notifikasi ke email (opsional)

#### Menutup Semua Sesi Lain

1. **Klik "End All Other Sessions"**
   - Di halaman session management
   - Atau di menu dropdown profil
2. **Konfirmasi dengan Password**

   ```
   ⚠️ End All Other Sessions?
   
   Are you sure you want to end all sessions except the current one?
   This action will force logout all other devices connected to your account.
   
   Password: [**********]
   
   [ Cancel ]  [ End All Sessions ]
   ```
3. **Hasil**
   - ✅ Semua device lain logout
   - 🔒 Hanya device saat ini yang tetap login
   - 🔐 Semua aplikasi terhubung tetap aktif

## 🛠️ Mengatasi Masalah Umum

### ❌ Masalah Login

#### Tidak Bisa Masuk ke Akun

**🔍 Diagnosis Masalah:**

1. **Periksa Kredensial**

   ```
   ❌ Email salah → Pastikan email tepat (cek typo)
   ❌ Password salah → Coba ketik ulang perlahan
   ❌ Caps Lock aktif → Pastikan capslock off
   ```
2. **Cek Status Akun**
   - ⚠️ Akun belum diverifikasi → Cek email verifikasi
   - 🚫 Akun diblokir → Hubungi customer support
   - 🔒 2FA bermasalah → Gunakan recovery code

**🛠️ Solusi Step-by-Step:**

1. **Reset Password**
   - Klik "Forgot Password?" di halaman login
   - Masukkan email → cek inbox → klik link reset
2. **Verifikasi Email**
   - Cek folder spam/junk untuk email verifikasi
   - Klik "Resend Verification" jika perlu
3. **Clear Browser Data**
   - Hapus cache dan cookies
   - Coba mode incognito/private
   - Test dengan browser berbeda

#### Masalah 2FA

**🔐 Authenticator App Tidak Berfungsi:**

1. **Sinkronisasi Waktu**

   ```
   Android: Settings → Date & Time → Automatic
   iPhone: Settings → General → Date & Time → Set Automatically
   ```
2. **Regenerate Kode**
   - Hapus account di authenticator app
   - Setup ulang dengan QR code baru
3. **Gunakan Recovery Code**
   - Klik "Use Recovery Code" saat login
   - Masukkan salah satu dari 8 kode yang disimpan

**📧 Email 2FA Bermasalah:**

1. **Cek Email**
   - Periksa folder inbox, spam, junk
   - Tunggu maksimal 5 menit
2. **Resend Code**
   - Klik "Resend code" (max 3x per hari)
   - Jika mentok limit, tunggu 24 jam
3. **Verifikasi Email Aktif**
   - Pastikan email masih bisa menerima pesan
   - Coba kirim email test ke email sendiri

### 📧 Masalah Email

#### Email Verifikasi Tidak Diterima

**🔍 Langkah Troubleshooting:**

1. **Cek Semua Folder**

   ```
   📥 Inbox (kotak masuk)
   🗑️ Spam/Junk
   📂 Promotional/Updates (Gmail)
   📁 Social (Gmail)
   ```
2. **Verifikasi Email Address**
   - Pastikan email diketik dengan benar
   - Cek tidak ada spasi di awal/akhir
   - Pastikan domain email valid (.com, .id, dll)
3. **Whitelist Domain**
   - Tambahkan `@mubarokah.com` ke contact
   - Atau tambahkan ke safe sender list
4. **Resend Email**
   - Klik "Resend Verification Email"
   - Tunggu 5-10 menit
   - Maksimal 3x resend per 24 jam

#### Email Notifikasi Berlebihan

1. **Atur Preferensi**

   ```
   Profil → Settings → Notification Preferences
   ```
2. **Unsubscribe**
   - Klik link "Unsubscribe" di email yang tidak diinginkan
   - Atau reply dengan "STOP"

### 🔄 Masalah Akses Aplikasi

#### Tidak Bisa Mengakses Aplikasi Terintegrasi

**🔍 Diagnosis:**

1. **Cek Izin Aplikasi**

   ```
   Dashboard → Manage Sessions → Connected Applications
   ```
   - Pastikan aplikasi ada dalam daftar
   - Periksa scope yang diberikan
2. **Cek Session**
   - Pastikan masih login di Mubarokah ID
   - Refresh halaman jika perlu

**🛠️ Solusi:**

1. **Re-authorize Aplikasi**
   - Logout dari aplikasi
   - Login ulang dengan Mubarokah ID
   - Berikan izin ulang jika diminta
2. **Clear Cache**
   - Hapus cache browser
   - Atau gunakan mode incognito
3. **Revoke & Re-grant Access**
   - Cabut akses aplikasi di session management
   - Akses aplikasi lagi dan beri izin ulang

#### Error "Access Denied"

**Penyebab Umum:**

- 🚫 Akun tidak memiliki akses ke aplikasi
- ⏰ Session expired
- 🔐 Izin dicabut oleh admin
- 🛡️ Aplikasi memerlukan approval khusus

**Solusi:**

1. **Hubungi Admin Aplikasi** untuk memberikan akses
2. **Re-login** untuk refresh session
3. **Cek Role** akun Anda (user/admin/super-admin)

### 🐛 Masalah Teknis

#### Website Lambat atau Error

**🔍 Quick Fixes:**

1. **Refresh Page**

   ```
   Windows: Ctrl + R atau F5
   Mac: Cmd + R
   Mobile: Pull down to refresh
   ```
2. **Hard Refresh**

   ```
   Windows: Ctrl + Shift + R
   Mac: Cmd + Shift + R
   ```
3. **Clear Browser Data**

   ```
   Chrome: Settings → Privacy → Clear browsing data
   Firefox: Settings → Privacy → Clear Data
   Safari: Develop → Empty Caches
   ```
4. **Try Incognito Mode**

   ```
   Chrome: Ctrl + Shift + N
   Firefox: Ctrl + Shift + P
   Safari: File → New Private Window
   ```

**🛠️ Advanced Troubleshooting:**

1. **Check Internet Connection**
   - Test dengan website lain
   - Restart router/modem
   - Ganti DNS ke 8.8.8.8 & 8.8.4.4
2. **Update Browser**
   - Pastikan browser versi terbaru
   - Enable automatic updates
3. **Disable Extensions**
   - Matikan AdBlock, VPN, atau extension lain
   - Test satu per satu untuk isolasi masalah
4. **Try Different Device/Browser**
   - Test dengan phone/tablet
   - Coba browser berbeda (Chrome, Firefox, Safari)

## 🔒 Tips Keamanan Terbaik

### 🛡️ Melindungi Akun Anda

#### Password Security

**✅ Kriteria Password Kuat:**

```
📏 Minimal 12 karakter (bukan 8!)
🔤 Kombinasi huruf besar & kecil
🔢 Mengandung angka
🔣 Mengandung simbol (!@#$%^&*()_+)
🚫 Bukan kata dalam kamus
🚫 Bukan informasi personal
🚫 Bukan pattern keyboard (qwerty123)
```

**🎯 Contoh Password Kuat:**

```
❌ Buruk: password123
❌ Buruk: mubarokah2024
❌ Buruk: namasaya1990
✅ Bagus: Mbk#2024!Secure
✅ Bagus: MyP@ssW0rd!2024
✅ Bagus: Buk4r@k#Aman2024
```

**🔧 Tools Rekomendasi:**

- 🥇 **Bitwarden** (gratis, open source)
- 🥈 **1Password** (premium, user-friendly)
- 🥉 **LastPass** (freemium)

#### Two-Factor Authentication

**🏆 Ranking Metode 2FA:**

1. 🥇 **Hardware Keys** (YubiKey) - paling aman
2. 🥈 **Authenticator Apps** - sangat aman, praktis
3. 🥉 **Email** - aman untuk kebanyakan orang
4. 🚫 **SMS** - TIDAK direkomendasikan (mudah diintercept)

**📱 Backup Strategy:**

- 📲 **Multiple Devices**: Setup authenticator di 2 device
- 💾 **Backup Codes**: Print dan simpan di safe
- 🔄 **Recovery Plan**: Siapkan plan jika HP hilang

#### Monitoring Aktivitas

**🔍 Hal yang Harus Diperhatikan:**

1. **Email Notifikasi Keamanan**

   ```
   📧 Login dari lokasi baru
   📧 Password diubah
   📧 2FA diaktifkan/dinonaktifkan

## 🔐 Mengamankan Akun dengan 2FA

**Two-Factor Authentication (2FA)** adalah lapisan keamanan tambahan yang **SANGAT DIREKOMENDASIKAN**. Tanpa 2FA, akun Anda 10x lebih rentan diretas\!

### 🛡️ Mengapa 2FA Penting?

- 🚫 **Melindungi dari pencurian password**
- 🚫 **Mencegah akses tidak sah meski password bocor**
- 📊 **Mengurangi risiko peretasan hingga 99.9%**
- ✅ **Memberikan notifikasi login yang mencurigakan**

### 📱 Metode 1: Setup 2FA dengan Aplikasi Authenticator

#### Persiapan

1. **Download Aplikasi Authenticator** (pilih salah satu):
   - 🥇 **Google Authenticator** (paling populer)
   - 🥈 **Authy** (backup cloud)
   - 🥉 **Microsoft Authenticator** (integrasi Microsoft)

#### Langkah Setup

1. **Masuk ke Pengaturan Keamanan**

   ```
   Dashboard → Profil → Keamanan → Setup Two-Factor Authentication
   ```
2. **Pilih "Setup Authenticator"**
   - Klik opsi Authenticator App
   - Muncul halaman dengan QR Code
3. **Scan QR Code**
   - Buka aplikasi authenticator
   - Pilih "Add Account" atau ikon "\+"
   - Scan QR code di layar
   - **Alternatif**: Masukkan kode manual jika tidak bisa scan
4. **Verifikasi Setup**
   - Aplikasi akan menampilkan kode 6 digit
   - Masukkan kode di website
   - Klik "Verifikasi & Aktifkan"
5. **Simpan Recovery Codes** ⚠️ **SANGAT PENTING\!**
   - Download atau print 8 recovery codes
   - Simpan di tempat aman (tidak di HP/komputer)
   - Kode ini untuk akses darurat jika HP hilang

### 📧 Metode 2: Setup 2FA dengan Email

#### Langkah Setup

1. **Pilih "Setup Email"**
   - Di halaman 2FA setup, pilih Email
   - Sistem kirim kode ke email terdaftar
2. **Cek Email**
   - Buka email dari "Mubarokah ID Security Team"
   - Cari kode 6 digit
   - **Jika tidak ada**: Cek folder spam/junk
3. **Masukkan Kode**
   - Input kode 6 digit di website
   - Klik "Verifikasi & Aktifkan"
4. **Simpan Recovery Codes**
   - Download 8 recovery codes
   - Simpan dengan aman

### 🔄 Cara Menggunakan 2FA Saat Login

**Setelah 2FA aktif, setiap login akan memerlukan:**

1. **Login Normal**
   - Masukkan email \+ password
   - Klik "Masuk"
2. **Verifikasi 2FA**
   - **Jika pakai Authenticator**: Buka app, masukkan kode 6 digit
   - **Jika pakai Email**: Cek email, masukkan kode 6 digit
3. **Akses Berhasil**
   - Masuk ke dashboard

### 🆘 Menggunakan Recovery Code

**Jika HP hilang atau tidak bisa akses authenticator:**

1. Di halaman 2FA, klik "Use Recovery Code"
2. Masukkan salah satu dari 8 recovery codes
3. **PENTING**: Setiap code hanya bisa dipakai 1x
4. Setelah login, segera setup 2FA ulang

### ❌ Menonaktifkan 2FA

> ⚠️ **Peringatan**: Menonaktifkan 2FA mengurangi keamanan akun\!

1. **Masuk ke Pengaturan**

   ```
   Profil → Keamanan → Two-Factor Authentication
   ```
2. **Klik "Disable 2FA"**
   - Masukkan password untuk konfirmasi
   - Klik "Disable Two-Factor Authentication"
3. **2FA Berhasil Dinonaktifkan**
   - Akun kembali ke keamanan dasar
   - Recovery codes dihapus

## 🛠️ Mengatasi Masalah Umum

### ❌ Masalah Login

#### Tidak Bisa Masuk ke Akun

**🔍 Diagnosis Masalah:**

1. **Periksa Kredensial**

   ```
   ❌ Email salah → Pastikan email tepat (cek typo)
   ❌ Password salah → Coba ketik ulang perlahan
   ❌ Caps Lock aktif → Pastikan capslock off
   ```
2. **Cek Status Akun**
   - ⚠️ Akun belum diverifikasi → Cek email verifikasi
   - 🚫 Akun diblokir → Hubungi customer support
   - 🔒 2FA bermasalah → Gunakan recovery code

**🛠️ Solusi Step-by-Step:**

1. **Reset Password**
   - Klik "Forgot Password?" di halaman login
   - Masukkan email → cek inbox → klik link reset
2. **Verifikasi Email**
   - Cek folder spam/junk untuk email verifikasi
   - Klik "Resend Verification" jika perlu
3. **Clear Browser Data**
   - Hapus cache dan cookies
   - Coba mode incognito/private
   - Test dengan browser berbeda

#### Masalah 2FA

**🔐 Authenticator App Tidak Berfungsi:**

1. **Sinkronisasi Waktu**

   ```
   Android: Settings → Date & Time → Automatic
   iPhone: Settings → General → Date & Time → Set Automatically
   ```
2. **Regenerate Kode**
   - Hapus account di authenticator app
   - Setup ulang dengan QR code baru
3. **Gunakan Recovery Code**
   - Klik "Use Recovery Code" saat login
   - Masukkan salah satu dari 8 kode yang disimpan

**📧 Email 2FA Bermasalah:**

1. **Cek Email**
   - Periksa folder inbox, spam, junk
   - Tunggu maksimal 5 menit
2. **Resend Code**
   - Klik "Resend code" (max 3x per hari)
   - Jika mentok limit, tunggu 24 jam
3. **Verifikasi Email Aktif**
   - Pastikan email masih bisa menerima pesan
   - Coba kirim email test ke email sendiri

### 📧 Masalah Email

#### Email Verifikasi Tidak Diterima

**🔍 Langkah Troubleshooting:**

1. **Cek Semua Folder**

   ```
   📥 Inbox (kotak masuk)
   🗑️ Spam/Junk
   📂 Promotional/Updates (Gmail)
   📁 Social (Gmail)
   ```
2. **Verifikasi Email Address**
   - Pastikan email diketik dengan benar
   - Cek tidak ada spasi di awal/akhir
   - Pastikan domain email valid (.com, .id, dll)
3. **Whitelist Domain**
   - Tambahkan `@mubarokah.com` ke contact
   - Atau tambahkan ke safe sender list
4. **Resend Email**
   - Klik "Resend Verification Email"
   - Tunggu 5-10 menit
   - Maksimal 3x resend per 24 jam

#### Email Notifikasi Berlebihan

1. **Atur Preferensi**

   ```
   Profil → Settings → Notification Preferences
   ```
2. **Unsubscribe**
   - Klik link "Unsubscribe" di email yang tidak diinginkan
   - Atau reply dengan "STOP"

### 🔄 Masalah Akses Aplikasi

#### Tidak Bisa Mengakses Aplikasi Terintegrasi

**🔍 Diagnosis:**

1. **Cek Izin Aplikasi**

   ```
   Dashboard → Manage Sessions → Connected Applications
   ```
   - Pastikan aplikasi ada dalam daftar
   - Periksa scope yang diberikan
2. **Cek Session**
   - Pastikan masih login di Mubarokah ID
   - Refresh halaman jika perlu

**🛠️ Solusi:**

1. **Re-authorize Aplikasi**
   - Logout dari aplikasi
   - Login ulang dengan Mubarokah ID
   - Berikan izin ulang jika diminta
2. **Clear Cache**
   - Hapus cache browser
   - Atau gunakan mode incognito
3. **Revoke & Re-grant Access**
   - Cabut akses aplikasi di session management
   - Akses aplikasi lagi dan beri izin ulang

#### Error "Access Denied"

**Penyebab Umum:**

- 🚫 Akun tidak memiliki akses ke aplikasi
- ⏰ Session expired
- 🔐 Izin dicabut oleh admin
- 🛡️ Aplikasi memerlukan approval khusus

**Solusi:**

1. **Hubungi Admin Aplikasi** untuk memberikan akses
2. **Re-login** untuk refresh session
3. **Cek Role** akun Anda (user/admin/super-admin)

### 🐛 Masalah Teknis

#### Website Lambat atau Error

**🔍 Quick Fixes:**

1. **Refresh Page**

   ```
   Windows: Ctrl + R atau F5
   Mac: Cmd + R
   Mobile: Pull down to refresh
   ```
2. **Hard Refresh**

   ```
   Windows: Ctrl + Shift + R
   Mac: Cmd + Shift + R
   ```
3. **Clear Browser Data**

   ```
   Chrome: Settings → Privacy → Clear browsing data
   Firefox: Settings → Privacy → Clear Data
   Safari: Develop → Empty Caches
   ```
4. **Try Incognito Mode**

   ```
   Chrome: Ctrl + Shift + N
   Firefox: Ctrl + Shift + P
   Safari: File → New Private Window
   ```

**🛠️ Advanced Troubleshooting:**

1. **Check Internet Connection**
   - Test dengan website lain
   - Restart router/modem
   - Ganti DNS ke 8.8.8.8 & 8.8.4.4
2. **Update Browser**
   - Pastikan browser versi terbaru
   - Enable automatic updates
3. **Disable Extensions**
   - Matikan AdBlock, VPN, atau extension lain
   - Test satu per satu untuk isolasi masalah
4. **Try Different Device/Browser**
   - Test dengan phone/tablet
   - Coba browser berbeda (Chrome, Firefox, Safari)

```

**🎯 Contoh Password Kuat:**

```
❌ Buruk: password123
❌ Buruk: mubarokah2024
❌ Buruk: namasaya1990

- 🥈 **1Password** (premium, user-friendly)
- 🥉 **LastPass** (freemium)

**🏆 Ranking Metode 2FA:**

1. 🥇 **Hardware Keys** (YubiKey) - paling aman
2. 🥈 **Authenticator Apps** - sangat aman, praktis
3. 🥉 **Email** - aman untuk kebanyakan orang
4. 🚫 **SMS** - TIDAK direkomendasikan (mudah diintercept)

**📱 Backup Strategy:**

- 📲 **Multiple Devices**: Setup authenticator di 2 device
- 💾 **Backup Codes**: Print dan simpan di safe
- 🔄 **Recovery Plan**: Siapkan plan jika HP hilang

#### Monitoring Aktivitas

1. **Email Notifikasi Keamanan**

2. **Session Management**

   ```
   👀 Cek device yang login minimal seminggu sekali
   🌍 Perhatikan lokasi yang tidak dikenal
   ⏰ Waspada aktivitas di jam aneh
   ```
3. **Connected Apps**

   ```
   🔍 Review aplikasi terhubung setiap bulan

   ⚠️ Perhatikan scope permission yang berlebihan

### 🚨 Mengenali Ancaman

#### Phishing Attacks

**🎣 Ciri-ciri Email/Pesan Phishing:**

```
⚠️ Subject mendesak: "Akun akan ditutup dalam 24 jam!"
⚠️ From address mencurigakan: mubarokah-id.security@gmail.com
⚠️ Link palsu: mubarokah-iid.com (perhatikan typo)
⚠️ Request password: "Reply dengan password untuk verifikasi"
⚠️ Grammar buruk: "Anda account sudah di hack"
```

**🛡️ Cara Melindungi Diri:**

1. **Selalu cek sender** dengan teliti
2. **Hover link** sebelum klik (lihat URL asli)
3. **Akses langsung** ke website resmi, jangan klik link
4. **Verifikasi** melalui channel resmi jika ragu

#### Social Engineering

**📞 Skenario Umum:**

```
☎️ "Saya dari IT Mubarokah ID, ada masalah dengan akun Anda..."
💬 "Kirim OTP untuk verifikasi identitas..."
📱 "Download aplikasi ini untuk update keamanan..."
💰 "Anda menang hadiah, konfirmasi dengan password..."
```

**🚫 Yang TIDAK BOLEH Dilakukan:**

- ❌ Memberikan password via telepon/chat
- ❌ Share OTP/kode 2FA kepada siapapun
- ❌ Download software dari link tidak resmi
- ❌ Transfer uang untuk "verifikasi"

#### Malware & Virus

**🦠 Tanda-tanda Terinfeksi:**

- 🐌 Computer sangat lambat
- 🎉 Pop-up iklan berlebihan
- 🔄 Browser homepage berubah sendiri
- 📧 Email terkirim tanpa sepengetahuan
- 💳 Transaksi tidak dikenal

**🛡️ Pencegahan:**

1. **Antivirus Premium**: Windows Defender \+ Malwarebytes
2. **Browser Security**: uBlock Origin, HTTPS Everywhere
3. **OS Updates**: Selalu update Windows/Mac/Linux
4. **Safe Browsing**: Hindari situs mencurigakan

### 📱 Keamanan Device

#### Computer/Laptop Security

**🔒 Basic Security:**

```
🔐 Screen lock password (bukan PIN sederhana)
⏰ Auto-lock setelah 5 menit idle
🔄 Automatic OS updates
🛡️ Real-time antivirus protection
🔥 Firewall aktif

**🗂️ Data Protection:**

```
💾 Backup reguler (cloud + external drive)

**📱 Essential Settings:**

```
🔐 Strong screen lock (PIN/Password/Biometric)
🔄 Auto-update apps dan OS
📍 Find My Device aktif
🔒 App permissions terbatas
🚫 Unknown sources dinonaktifkan

**🛡️ Advanced Protection:**

```
📲 VPN untuk WiFi publik

📱 Anti-theft apps

💾 Regular backup
