## 🔐 Two-Factor Authentication (2FA)

> 💡 **Mengapa 2FA penting?** Melindungi dari pencurian password, mencegah akses tidak sah, dan menurunkan risiko peretasan secara signifikan.

**Langkah aktivasi (Authenticator App – disarankan):**

1. **Unduh aplikasi**: Google Authenticator / Authy / Microsoft Authenticator
2. **Buka pengaturan keamanan**: Dashboard → Profil → Keamanan → **Setup 2FA**
3. **Pindai QR / masukkan kode manual** di aplikasi authenticator
4. **Verifikasi**: Masukkan **kode 6 digit** → **Aktifkan**
5. **Simpan recovery codes**: Unduh/print **8 kode** dan simpan **offline**

**Alternatif (2FA via Email):** Pilih **Email** saat setup 2FA → masukkan kode 6 digit yang dikirim ke email → **Aktifkan** → simpan recovery codes.

> ⚠️ **Menonaktifkan 2FA mengurangi keamanan akun.** Lakukan hanya bila diperlukan dan pahami risikonya.

## 👤 Profil & Akun

**Akses:** Dashboard → Profil → **Edit Profile**

- Ubah: **Nama lengkap**, **Username**, **Email**, **No. WhatsApp**
- Lengkapi: **Bio**, **Tempat/Tanggal Lahir (DD/MM/YYYY)**, **Jenis Kelamin**, **Alamat**
- Klik **Update Profile**

**Ubah Password:** Profil → Keamanan → **Password Security** **Reset (lupa password):** klik **Forgot Password?** (link valid **1 jam**)

**Hapus Akun (permanen):** Profil → **Danger Zone** → **Delete My Account** → konfirmasi password.

## 🔗 Aplikasi Terhubung

**Cara akses:**

- Dari **Dashboard** Mubarokah ID → pilih aplikasi → diarahkan otomatis (SSO)
- Atau dari situs aplikasi → **Login with Mubarokah ID**

**Otorisasi & izin akses (contoh):**

- Profil dasar (nama, email, foto)
- Informasi kontak (email, nomor HP)
- Data aplikasi spesifik / _scope_ tertentu

**Cabut akses aplikasi:** Dashboard → **Manage Sessions** → **Connected Apps** → pilih aplikasi → **Revoke Access**.

## 💻 Keamanan & Sesi

**Pantau sesi aktif:** Dashboard → **Manage Sessions** → **Active Devices** Lihat perangkat, browser, lokasi (IP), aktivitas terakhir, IP, waktu login.

- **Akhiri sesi tertentu:** **End Session**
- **Paksa keluar semua kecuali saat ini:** **End All Other Sessions** (butuh password)

> 🔎 **Tips:** Tinjau daftar perangkat secara berkala untuk mendeteksi akses tak dikenal.

## 🛠️ Troubleshooting Umum

**Masalah Login**

- Pastikan email & password benar (cek **Caps Lock**)
- Cek status verifikasi akun
- Gunakan **Forgot Password** untuk reset

**Masalah 2FA**

- Sinkronkan waktu perangkat (Android/iOS)
- Setup ulang authenticator bila perlu
- Gunakan **Recovery Code** jika kehilangan akses

**Masalah Email**

- Periksa **Inbox/Spam/Promotions/Social**
- _Whitelist_ `@mubarokah.com`
- Gunakan **Resend Verification** (maks. 3×/24 jam)

**Akses Aplikasi**

- Pastikan aplikasi terhubung & scope sesuai
- Re-authorize / bersihkan cache / revoke & re-grant
- Cek role/izin admin; sesi bisa saja **expired**

## 🔒 Tips Keamanan

**Password Aman**

- Minimal **12 karakter** dengan huruf besar/kecil, angka, simbol
- Hindari data pribadi dan pola umum (`qwerty123`)

**Gunakan Password Manager**

- Bitwarden • 1Password • LastPass

**Metode 2FA (urutan rekomendasi)**

1. **Hardware Key (YubiKey)**
2. **Authenticator App**
3. **Email**
4. **SMS** _(tidak disarankan)_

**Monitoring**

- Aktifkan notifikasi: login lokasi baru, ubah password/2FA, penggunaan recovery code
- Tinjau **Connected Apps** bulanan & **Active Sessions** mingguan

**Keamanan Perangkat & Jaringan**

- Komputer: kunci layar, auto-lock, update OS, antivirus real-time, firewall
- Mobile: kunci kuat/biometrik, auto-update, Find My Device, batasi izin aplikasi
- Jaringan: Wi-Fi WPA3, nonaktifkan WPS, update firmware router, gunakan **VPN** di Wi-Fi publik
