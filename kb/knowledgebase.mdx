## 🚀 Getting Started

### 📝 Create a New Account

1. **Open the registration page** Visit [**accounts.mubarokah.com**](http://accounts.mubarokah.com) and click **Sign Up**.
2. **Fill in your details** Full Name, Active Email, WhatsApp Number (+62), Password (min. 8 characters), Confirm Password.
3. **Choose a verification method** **Recommended:** **Email verification** (more secure). Alternatively, use **WhatsApp**.
4. **Complete verification & log in** Follow the verification instructions → return to login → enter your credentials → click **Sign In**.

> **Password Strength Example** 🔴 Weak: `mubarokah` • 🟡 Medium: `mubarokah123` • 🟢 Strong: `Mubarokah123!`

## 🔐 Two-Factor Authentication (2FA)

> 💡 **Why 2FA matters:** It protects against password theft, blocks unauthorized access, and reduces hacking risk by over 99%.

**Setup via Authenticator App (Recommended):**

1. **Download an authenticator app:** Google Authenticator, Authy, or Microsoft Authenticator.
2. **Open Security Settings:** Dashboard → Profile → Security → **Setup 2FA**.
3. **Scan the QR code** or manually enter the secret key in your app.
4. **Verify:** Enter the 6-digit code → click **Activate**.
5. **Save your recovery codes:** Download/print all 8 and store them safely offline.

**Alternative (2FA via Email):** Select **Email** in 2FA setup → enter the 6-digit code sent to your inbox → **Activate** → save your recovery codes.

> ⚠️ **Disabling 2FA reduces account security.** Only do so if absolutely necessary.

## 👤 Managing Your Profile & Account

**Access:** Dashboard → Profile → **Edit Profile**

- Update your **Full Name**, **Username**, **Email**, and **WhatsApp Number**
- Complete additional info: **Bio**, **Birthplace/Date (DD/MM/YYYY)**, **Gender**, **Address**
- Click **Update Profile**

**Change Password:** Profile → Security → **Password Security** **Reset (Forgot Password):** click **Forgot Password?** (link valid for **1 hour**)

**Delete Account (Permanent):** Profile → **Danger Zone** → **Delete My Account** → confirm with password.

## 🔗 Connected Applications

**How to Access:**

- From the **Mubarokah ID Dashboard** → select the desired app → auto-redirected via SSO.
- Or from the app’s website → click **Login with Mubarokah ID**.

**Authorization & Permissions (Example):**

- Basic Profile (name, email, photo)
- Contact Information (email, phone number)
- Application-specific data (_scope_)

**Revoke Application Access:** Dashboard → **Manage Sessions** → **Connected Apps** → select app → click **Revoke Access**.

## 💻 Security & Sessions

**View Active Sessions:** Dashboard → **Manage Sessions** → **Active Devices** Displays device type, browser, IP location, last activity, and login time.

- **End a specific session:** Click **End Session**.
- **Log out all other devices:** Click **End All Other Sessions** (requires password).

> 🔎 **Tip:** Review active devices regularly to detect suspicious logins early.

## 🛠️ Common Troubleshooting

**Login Issues**

- Verify your email and password (check **Caps Lock**)
- Confirm account verification status
- Use **Forgot Password** to reset

**2FA Problems**

- Sync device time (Android/iOS)
- Re-setup the authenticator app
- Use a **Recovery Code** if you’ve lost access

**Email Issues**

- Check **Inbox/Spam/Promotions/Social** folders
- Whitelist `@mubarokah.com`
- Use **Resend Verification** (max. 3 times per 24h)

**App Access Issues**

- Ensure app permissions (_scopes_) are correct
- Re-authorize, clear cache, or revoke/reconnect access
- Confirm your account’s role (User/Admin); expired sessions may cause denial

## 🔒 Security Best Practices

**Strong Passwords**

- At least **12 characters** with uppercase, lowercase, numbers, and symbols
- Avoid personal data and common patterns (`qwerty123`)

**Recommended Password Managers**

- Bitwarden • 1Password • LastPass

**Preferred 2FA Methods**

1. **Hardware Key (YubiKey)**
2. **Authenticator App**
3. **Email**
4. **SMS** _(not recommended)_

**Monitoring Tips**

- Enable alerts for: new device login, password/2FA changes, recovery code usage
- Review **Connected Apps** monthly & **Active Sessions** weekly

**Device & Network Security**

- PC: screen lock, auto-lock, OS updates, real-time antivirus, firewall
- Mobile: strong PIN/biometric, auto-update, Find My Device, app permission control
- Network: WPA3 Wi-Fi, disable WPS, router firmware updates, use **VPN** on public Wi-Fi
